# RURAL INFRASTRUCTURE INSPECTION & MONITORING FRAMEWORK
## (Sierra Leone – Field Engineer Guide)

---

# 1. VISIT METADATA (MANDATORY)

- Date & Time of Visit  
- Project Name  
- Project Type (Road, Building, Borehole, Bridge, etc.)  
- GPS Coordinates  
- Inspector Name & Team  
- Contractor Name  
- Site Supervisor Present  
- Weather Conditions  
- Visit Type:
  - Routine
  - Follow-up
  - Final
  - Emergency  

---

# 2. VISUAL EVIDENCE (MANDATORY)

- Geotagged Photos  
- Before / During / After Images  
- Videos (optional)  
- Drone Images (if available)  

### Capture:
- Work progress  
- Defects/issues  
- Materials  
- Workforce  

---

# 3. PHYSICAL PROGRESS TRACKING

- Overall % Completion  
- Component-level progress  
- Work completed since last visit  
- Planned vs Actual progress  
- Timeline Status:
  - On Schedule
  - Delayed (with reason)  

---

# 4. MILESTONE TRACKING (CRITICAL)

## A. Milestone Definition (Predefined per Project)

### Feeder Roads
- Site clearing completed  
- Earthworks completed  
- Sub-base laid  
- Drainage completed  
- Final compaction  

### Buildings
- Foundation completed  
- Walling to lintel  
- Roofing completed  
- Plastering completed  
- Finishing  

### Boreholes + Solar
- Drilling completed  
- Casing installed  
- Pump installed  
- Solar installed  
- Tank & distribution completed  

### Bridges / Culverts
- Excavation completed  
- Foundation works  
- Structural works  
- Decking  
- Approach roads  

### Grain Stores / Drying Floors
- Foundation completed  
- Structure erected  
- Flooring completed  
- Roofing completed  
- Finishing  

### Public Toilets
- Pit/septic completed  
- Structure built  
- Roofing completed  
- Fixtures installed  
- Ready for use  

---

## B. Milestone Logging (Every Visit)

For each milestone:

- Milestone Name  
- Planned Completion Date  
- Status:
  - Not Started
  - In Progress
  - Completed  
- % Completion  
- Date Achieved  
- Delay (days)  
- Reason for Delay  

---

## C. Milestone Verification

- Physical inspection required  
- Must match drawings/specifications  
- No acceptance without evidence  

---

## D. Milestone Evidence

- Before & After Photos  
- Close-up technical images  
- GPS-tagged proof  
- Optional video  

---

## E. Milestone Alerts

- Delayed milestone  
- Poor quality completion  
- Skipped milestone  
- Rework required  

---

## F. Milestone-Linked Actions

- Action created  
- Responsible person assigned  
- Deadline set  
- Follow-up required  

---

## G. Milestone Performance Indicators

- % completed on time  
- Average delay  
- Rework rate  
- Quality pass rate
